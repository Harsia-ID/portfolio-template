## ğŸ”§ Kustomisasi

Anda dapat dengan mudah mengkustomisasi portofolio ini:

- **Informasi Pribadi:** Ubah teks di bagian `Tentang Saya`, `Keahlian`, dan `Kontak` langsung di file `index.html`.
